GameDemo Backend - Công cụ quản lý dữ liệu

Quản lý dữ liệu game

Công cụ quản lý dữ liệu cho game của bạn. Chọn loại dữ liệu để xem và quản lý.

Loại dữ liệu:

Hero Equipment Player Level Shop Zone Drop Resource Zone Reward

API Endpoints

Các API endpoints có sẵn:

Upload Data

API Endpoint Phương thức Mô tả
/api/uploadHeroData POST Upload dữ liệu Hero
/api/uploadHeroFile POST Upload file Hero JSON
/api/uploadEquipmentData POST Upload dữ liệu Equipment
/api/uploadEquipmentFile POST Upload file Equipment JSON
/api/uploadPlayerLevelData POST Upload dữ liệu Player Level
/api/uploadPlayerLevelFile POST Upload file Player Level JSON
/api/uploadShopData POST Upload dữ liệu Shop
/api/uploadShopFile POST Upload file Shop JSON
/api/uploadZoneData POST Upload dữ liệu Zone
/api/uploadZoneFile POST Upload file Zone JSON
/api/uploadDropResourceData POST Upload dữ liệu Drop Resource
/api/uploadDropResourceFile POST Upload file Drop Resource JSON
/api/uploadZoneRewardData POST Upload dữ liệu Zone Reward
/api/uploadZoneRewardFile POST Upload file Zone Reward JSON

Get Data

API Endpoint Phương thức Mô tả
/api/getHeroData GET Lấy dữ liệu Hero
/api/getEquipmentData GET Lấy dữ liệu Equipment
/api/getPlayerLevelData GET Lấy dữ liệu Player Level
/api/getShopData GET Lấy dữ liệu Shop
/api/getZoneData GET Lấy dữ liệu Zone
/api/getDropResourceData GET Lấy dữ liệu Drop Resource
/api/getZoneRewardData GET Lấy dữ liệu Zone Reward

Hướng dẫn thêm loại dữ liệu mới

Để thêm một loại dữ liệu mới, bạn cần:

  1. Định nghĩa model trong src/models/gameDataModels.js
  2. Tạo trang quản lý dữ liệu trong thư mục public
  3. Sử dụng các API endpoints theo định dạng /api/upload[Type]Data, /api/upload[Type]File, /api/get[Type]Data
